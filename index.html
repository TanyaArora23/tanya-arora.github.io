Projects

Cyclic Pursuit with Crazyflie Robots

Developed a multi-agent Cyclic Pursuit strategy using Crazyflie drones and ROS 2. Synchronized drone states and implemented velocity control using Crazyswarm2 + CrazySim in Gazebo.

Object Avoidance with Turtlebot4

Contributed to an autonomous exploration system using a fleet of Turtlebot4 robots in unknown environments, leveraging odometry and collision avoidance strategies.

RB5 Robot Navigation Pipeline

Designed a full perception-localization-mapping-control pipeline for Qualcomm RB5 robots to navigate arenas using AprilTags and path optimization techniques.

Orientation Estimation from IMU

Implemented a projected gradient descent algorithm using IMU data to estimate 3D orientation and generate panoramic images from camera data.

SLAM Comparison Study

Performed comparative analysis of SLAM using encoder/IMU data vs. Lidar-based ICP, evaluating performance and map accuracy.
