Cyclic Pursuit with Crazyflie Robots

Developed a velocity-based cyclic pursuit controller for multi-agent coordination using Crazyflie drones. Implemented synchronization via ROS 2 and simulated in Gazebo using Crazyswarm2 and CrazySim.

  • Multi-agent coordination
  • ROS 2 velocity control
  • Gazebo + CrazySim simulation

Object Avoidance with Turtlebot4

Designed and tested an autonomous exploration strategy using multiple Turtlebot4 robots with odometry-based localization and real-time collision avoidance.

  • Odometry navigation
  • Obstacle detection and avoidance
  • Multi-robot exploration

RB5 Robot Navigation Pipeline

Built a modular navigation pipeline for Qualcomm RB5 robots incorporating AprilTag detection, trajectory planning, and map-aware localization.

  • AprilTag-based localization
  • Path planning and optimization
  • System integration on RB5

Orientation Estimation from IMU

Estimated 3D orientation trajectories from IMU data using projected gradient descent on unit quaternions. Generated panoramic views from camera images aligned using estimated orientations.

  • IMU calibration and filtering
  • Quaternion-based optimization (JAX)
  • Panoramic image generation

SLAM Comparison Study

Conducted a comparative evaluation between odometry-based and Lidar-ICP based SLAM algorithms in terms of accuracy and robustness.

  • SLAM pipeline benchmarking
  • ICP vs Odometry error analysis
  • Map quality evaluation
